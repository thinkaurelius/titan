Aurelius Titan

Titan Presentations

  • Titan: The Rise of Big Graph Data (presentation): An Introduction to graphs, graph algorithms, and the Titan graph database. Also includes an overview of the graph database landscape. (June 2012, hosted by Jive Software)
  • Titan: A Highly Scalable, Distributed Graph Database (poster): A presentation of an initial stress test evaluation of Titan using real Twitter network data. (July 2012, presented at GraphLab 2012 conference)
  • Titan Provides Real-Time Big Graph Data (article): A thorough evaluation of Titan based on a social network benchmark using real Twitter network data. (August 2012)
  • Titan: Big Graph Data with Cassandra (video, slides): Presentation of graph database use cases and Titan. Discusses some of Titan’s unique features and how Titan uses Cassandra to scale. (August 2012, presented at Cassandra 2012 Summit)
  • Big Graph Data: Understanding a Complex World (presentation): Solving 21st century problems with graphs and an overview of Titan and Faunus. (November 2012, presented at Bay Area Graph Geeks)
  • Data – Information – Knowledge: Adding Value through Graph Analysis (presentation): Discusses how graph analysis can add value to information and new advancements in Titan and Faunus. (March 2013, presented at Ebay)
  • Educating the Planet with Pearson Education (article): Presents a large-scale benchmark of Titan with a 121 billion edge graph. (May 2013, in collaboration with Pearson Education)
